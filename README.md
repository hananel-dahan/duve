# Duve

## Project setup

# Server

- `npm install`

- create DB schema named `duve_db`

- `npm start`

- update the .env file

- run post command to `localhost:3000/api/reservations/initialize` in postman

# Client

- `npm install`
<!-- - `npm run serve` to start the Frontend -->
- `npm run watch` to start the Frontend in dev mode
